## Gameplay
El Gameplay esta desarrollado con <a href="https://pixijs.com">Pixi.js</a>, combina un enfoque educativo y lúdico, ideal para todas las edades. El juego no solo pone a prueba tus conocimientos, sino que también enseña y refuerza conceptos matemáticos mediante explicaciones claras y ejemplos prácticos, adaptándose al nivel del jugador.

## [Usage](https://github.com/jmelendez07/MathG) 

### Desarrollo
¿Quieres contribuir? Genial!

Para corregir un error o mejorar un módulo existente, sigue estos pasos:

- Haz un fork del repositorio
- Crea una nueva rama (`git checkout -b improve-feature`)
- Realiza los cambios necesarios en los archivos
- Añade los cambios para reflejar las modificaciones realizadas
- Confirma tus cambios (`git commit -am 'Improve feature'`)
- Sube los cambios a la rama (`git push origin improve-feature`)
- Crea una Pull Request

### Bugs / Nuevas funcionalidades

Si encuentra algún error (el sitio web no pudo procesar la consulta o devolvió resultados inesperados), por favor, abra un ticket [aquí](https://github.com/jmelendez07/MathG/issues/new) incluyendo su consulta de búsqueda y el resultado esperado.

Si desea solicitar una nueva función, puede hacerlo abriendo un ticket [aquí](https://github.com/jmelendez07/MathG/issues/new). Incluya ejemplos de consultas y sus resultados correspondientes.


## Construido con

- [React](https://react.dev) - React te permite crear interfaces de usuario a partir de piezas individuales llamadas componentes. Crea tus propios componentes de React, como Miniatura, Botón de Me Gusta y Vídeo. Luego, combínalos para crear pantallas, páginas y aplicaciones completas.
- [Laravel](https://laravel.com/) - Laravel proporciona un ecosistema completo para desarrolladores web. Este framework PHP de código abierto, productos, paquetes y kits de inicio ofrecen todo lo necesario para crear, implementar y monitorizar aplicaciones web.
- [Inertia](https://inertiajs.com/) - Inertia te permite crear aplicaciones web de una sola página (SPA) completamente renderizadas en el cliente, sin la complejidad de las SPA modernas. Lo consigue aprovechando patrones del lado del servidor que ya conoces y te resultan útiles.
- [Pixi](https://pixijs.com/) - PixiJS es un motor de renderizado 2D avanzado y de código abierto diseñado para crear experiencias visuales impresionantes en la web.
- [Tailwind](https://tailwindcss.com/) - Tailwind CSS funciona escaneando todos tus archivos HTML, componentes JavaScript y cualquier otra plantilla en busca de nombres de clase, generando los estilos correspondientes y luego escribiéndolos en un archivo CSS estático.
- [MongoDB](https://www.mongodb.com/) - MongoDB es una base de datos operativa orientada a documentos, diseñada desde cero como alternativa a las bases de datos relacionales para aplicaciones modernas.
